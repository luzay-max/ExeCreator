# 恶搞启动器生成工具 v4.0 升级计划 (Roadmap)

基于 v3.0 版本（并发扫描引擎、自动签名的 Python GUI）的稳固基础，v4.0 将进行一次从底层到体验的全面重构，重点突破 **免杀能力**、**程序体积** 以及 **恶搞分发与反馈体验**。

---

## 🔴 高优先级任务 (核心重构与免杀)

### 1. Launcher 引擎底层重构 (Rust/C++)
- **文件**: `template/` [底层替换]
- **工作量**: 极高
- **内容**:
  - [ ] **技术选型**：验证使用 Rust 重写启动器扫描和跳转逻辑的可行性。
  - [ ] **资源注入机制**：实现 C++/Rust 预编译二进制文件（PE）的动态配置注入（如通过修改 Resource Section 或文件尾 Overlay）。
  - [ ] **剥离 PyInstaller**：彻底摆脱 Python 运行时依赖，目标生成物单文件体积 < 2MB（无膨胀情况下）。

### 2. 深度反制与免杀 (Anti-Analysis)
- **文件**: `template/launcher_core.rs` (新引擎)
- **工作量**: 高
- **内容**:
  - [ ] 实现反沙箱检测：检测 CPU 核心数（<2 退出）、内存大小、MAC 地址特征。
  - [ ] 实现反调试机制：调用 `IsDebuggerPresent()`、`CheckRemoteDebuggerPresent`，检测到分析环境则表现为良性应用（如打开记事本）。
  - [ ] 动态 API 解析：隐藏对系统敏感函数（如 `ShellExecute`、`MoveFileExW`）的直接调用。

---

## 🟡 中优先级任务 (生成器体验升级)

### 3. 现代化 Builder UI 重构
- **文件**: `builder/builder_gui.py`, `builder/widgets/`
- **工作量**: 中
- **内容**:
  - [ ] 引入 `customtkinter` 替换原生 `tkinter` 组件，实现现代化圆角界面。
  - [ ] 实现全局深色/浅色主题的自由切换。
  - [ ] 在配置界面添加“伪装加载界面”的实时预览面板（所见即所得）。

### 4. Webhook 战报回传机制
- **文件**: `builder/widgets/config_panel.py`, `template/`
- **工作量**: 中
- **内容**:
  - [ ] Builder 配置面板新增 "Webhook 追踪" 选项输入框。
  - [ ] Launcher 执行关键节点（如：未找到目标并成功跳转网页）时，发送异步 HTTP POST 请求。
  - [ ] 兼容常用推送服务：Server酱、钉钉机器人、飞书机器人格式。

---

## 🟢 低优先级任务 (附加功能与恶搞库拓展)

### 5. 一键云端分发模块
- **文件**: `builder/builder_core.py`, `builder/widgets/share_panel.py`
- **工作量**: 低
- **内容**:
  - [ ] 打包成功后，提供“生成分享链接/二维码”功能。
  - [ ] 接入公共临时文件分享 API（如奶牛快传、文叔叔），实现 EXE 自动上传并返回短链接。

### 6. 更多维度的恶搞载荷 (Payloads)
- **文件**: `template/payloads/`
- **工作量**: 低
- **内容**:
  - [ ] 增加系统级伪装错误弹窗（高保真蓝屏画面）。
  - [ ] 增加后台搞怪音频播放模块（可配置嵌入的 `.mp3` 或 `.wav`）。
  - [ ] 增加“鼠标短时间漂移”的整蛊选项。
