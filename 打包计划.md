# PrankLauncherBuilder 打包计划

## 一、项目概述

### 1.1 目标
将 `PrankLauncherBuilder` 项目完整打包为 Windows 可执行文件 (`.exe`)，实现：
- ✅ 无需安装 Python 即可运行
- ✅ 保持原有所有功能
- ✅ 提升用户体验（启动速度、界面优化）
- ✅ 解决当前已知问题

### 1.2 当前状态分析

#### 现有打包配置
| 组件 | 状态 | 问题 |
|------|------|------|
| `PrankLauncherBuilder.spec` | ✅ 已配置 | 依赖PyInstaller，需要Python环境 |
| `build.bat` | ✅ 已实现 | 仅打包生成器，非独立运行 |

#### 当前打包方式的问题
1. **运行环境依赖**：需要目标机器安装 Python 或便携版 Python
2. **启动速度**：PyInstaller 单文件模式首次启动较慢（解压过程）
3. **资源管理**：模板文件作为外部目录分发，路径问题多
4. **错误处理**：缺乏完整的异常捕获和用户友好的错误提示

    """错误处理装饰器"""

        except Exception as e:
            error_msg = f"发生错误: {str(e)}\n\n详细信息:\n{traceback.format_exc()}"

# builder/utils/cache.py
"""缓存管理"""
import json
import os
from pathlib import Path

class CacheManager:
    """构建缓存管理器"""
    
    def __init__(self, cache_file='.build_cache.json'):
        self.cache_file = cache_file
        self.cache = self._load_cache()
    
    def _load_cache(self):
        """加载缓存"""
        if os.path.exists(self.cache_file):
            try:
                with open(self.cache_file, 'r', encoding='utf-8') as f:
                    return json.load(f)
            except:
                return {}
        return {}
    
    def _save_cache(self):
        """保存缓存"""
        with open(self.cache_file, 'w', encoding='utf-8') as f:
            json.dump(self.cache, f, ensure_ascii=False, indent=2)
    
    def get(self, key):
        return self.cache.get(key)
    
    def set(self, key, value):
        self.cache[key] = value
        self._save_cache()
    
    def invalidate(self, key=None):
        """使缓存失效"""
        if key:
            self.cache.pop(key, None)
        else:
            self.cache.clear()
        self._save_cache()
